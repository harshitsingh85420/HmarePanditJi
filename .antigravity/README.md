# Antigravity Integration Setup

## Quick Start

### Option 1: Using Antigravity Desktop App (Recommended)

1. **Download Antigravity**: https://antigravity.google/download
2. **Install Chrome Extension**: Available in Antigravity settings
3. **Open your project** in Antigravity
4. **Enable Agent Skills**: Load `.antigravity/skills.json`
5. **Start testing**: Use visual testing skills

### Option 2: Using Gemini CLI

1. **Install**: `npm install -g @google/gemini-cli`
2. **Authenticate**: `gemini auth`
3. **Run tests**: `gemini -p "Test this app" --output-format json`

### Option 3: Visual Browser Testing (No Antigravity required)

1. **Run**: `pnpm visual:test --url http://localhost:3000`
2. **Check results**: `test-results/visual/`
3. **Tell Qwen Code**: "Check visual test results and fix issues"

## Configuration Files

| File | Purpose |

## Visual Testing Commands

```bash
# Run visual tests
pnpm visual:test --url http://localhost:3000

# Run with regression detection
pnpm visual:regression

# Custom test
node scripts/visual-test.js --url http://localhost:3000 --test "Login Flow"
```

## AI Testing Commands

```bash
# Run AI tests with Gemini CLI
pnpm ai:test "Find all bugs"

# Code review
pnpm ai:review

# Prepare fixes for Qwen Code
pnpm ai:fix
```

## Workflow: Antigravity → Qwen Code

1. **Antigravity runs tests** (visual or AI-powered)
2. **Results saved** to `test-results/`
3. **Tell Qwen Code**: "Check test-results/ and fix issues"
4. **Qwen Code analyzes** and provides fixes
5. **Apply fixes** and repeat

## Troubleshooting

### "Antigravity not found"
- Download from https://antigravity.google/download
- Or use Gemini CLI: `npm install -g @google/gemini-cli`

### "Gemini CLI auth failed"
- Run `gemini auth` and follow prompts
- Ensure you have a Google account with API access

### Visual tests failing
- Ensure app is running: `pnpm dev`
- Check URL is accessible
- Try headless mode: add `--headless` flag
